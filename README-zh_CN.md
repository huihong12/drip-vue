## 特性

- 提炼自企业级中后台产品的交互语言和视觉风格。
- 开箱即用的高质量 Vue 组件。

## 支持环境

- 现代浏览器和 IE9 及以上（需要 [polyfills](https://www.antdv.com/docs/vue/getting-started-cn/#兼容性)）。
- 支持服务端渲染。

## 安装

### 使用 npm 或 yarn 安装

```bash
$ npm install drip-vue --save
```

```bash
$ yarn add drip-vue
```

如果你的网络环境不佳，推荐使用 [cnpm](https://github.com/cnpm/cnpm)。
